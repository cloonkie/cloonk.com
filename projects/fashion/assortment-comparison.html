Assortment Comparison — Appendix

What this is

Assortment Comparison is a browser-based overlap tool for retailer assortment files. Upload a spreadsheet, choose retailers and brands, compare products by style/color/grid, and export the current view. The file never leaves your browser.

Quick start

1. Click Template for the expected columns.
2. Fill it in, then load it via Load Sheet or by dropping a file on the landing zone.
3. Pick up to five retailers, choose a comparison key, then inspect overlap segments and product rows.

Expected columns

Header matching is flexible; these names are the clearest path.

ColumnUse
RetailerRequired account/banner name used to build overlap sets.
BrandOptional brand filter and product table field.
ReleaseOptional release/season field.
StyleProduct style/model number. Selectable as the comparison key.
UPCProduct UPC/GTIN. Default comparison key; falls back to Style if blank.
ColorOptional color field shown in product details.
GridOptional grid/size field shown in product details.
Unit SoldSales unit metric.
Unit OHOn-hand inventory metric.

Glossary

Hover any term for a plain-language explanation.

TermWhat it means
OverlapA product key that appears in more than one selected retailer's assortment. Shared products across selected retailers.
KeyThe field used to decide whether rows represent the same product — either UPC or Style. The matching logic for product comparison.
Unit OHUnits on hand: inventory currently available or reported in the source file. Inventory count.
