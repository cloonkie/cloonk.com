Image Prep — Appendix

What this is

A browser-based batch image tool for prepping product and marketing imagery. Drop a set of images, choose how to resize them, set a background color behind any transparency, pick an output format, and download clean files — one by one or all at once.

How it works

Every image is decoded and redrawn on an HTML canvas with your chosen dimensions and background, then re-encoded to the format you select. Nothing is uploaded — all processing happens locally in your browser, so source files never leave your machine.

Steps

  1. Add images. Drop files onto the dropzone or click to browse. PNG, JPEG, WebP, GIF (first frame), BMP, and SVG are accepted.
  2. Set transform options. Resize, background color, and output format apply to the whole batch.
  3. Process & download. Review the output gallery, then download each file or the whole set.

Source images

Each accepted file is queued with a thumbnail and its native pixel dimensions. Remove individual items with the × button, or Clear all in the header.

Transform

Resize

  • Original — keep native dimensions.
  • Scale % — multiply both dimensions by a percentage.
  • Fit within — shrink to fit inside a max width × height, preserving aspect ratio. Optionally never enlarge smaller images.
  • Exact — force a precise canvas size. Contain fits the whole image and pads the rest with the background; Cover fills the frame and crops the overflow; Stretch distorts to the exact size.

Background — Transparent keeps the alpha channel (PNG/WebP only). Color fills behind transparent pixels and any padding. JPEG cannot store transparency, so it is always flattened onto the chosen color (white by default).

Reformat — Keep retains each file's original type where possible. PNG is lossless; JPEG and WebP expose a quality slider. A filename suffix and an optional _WIDTHxHEIGHT tag can be appended.

Output

Each result card shows the new preview on a checkerboard (so transparency is visible), the dimension change, the format, and the new file size with the percentage saved or added versus the source. Download individually or use Download all.

Tips

  • Set every product shot to a uniform square with Exact → Contain and a white background for a tidy line sheet or marketplace listing.
  • Use Fit within + WebP at ~80% quality to make web-ready assets that stay sharp but small.
  • GIFs are rendered from their first frame only — animation is not preserved.
  • Download all fires the saves in sequence; if your browser asks to allow multiple downloads, accept it.
  • Options apply to the whole batch and re-run automatically as you tweak them.
