Sell-Out Standardizer - Appendix

Standard Output

Each metric column becomes one row with source_file, source_sheet, source_row, report_period, cadence, year_basis, metric, and value. Product fields such as retailer, brand, style, color, SKU, UPC, size, and description remain as dimensions.

Stacked Headers

When a row above the metric headers contains timing labels such as WTD, MTD, QTD, or YTD, the app carries that cadence across repeated columns like Sale U and Sale $.

LY and ST%

The LY toggle treats LY and Last Year labels as comparison values instead of cadences. The ST% toggle adds calculated sell-through rows when Sale U and an on-hand unit column are available for the same cadence and year basis.

Sheets and Rows

Use the sheet checkboxes to include only the workbook tabs that belong in the combined output. Delete Rows removes 1-based Excel row numbers before parsing, so header positions shift. Ignore Rows keeps the sheet structure intact but skips those original row numbers in the final output.

Configurations

Exporting a config saves the detected rows, report period, default cadence, and current field mapping. Import it next week and the same retailer report shape can be normalized quickly.
